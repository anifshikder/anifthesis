Associative memory · Complex networks

Remembering by walking the network

Classic memory networks recall a pattern by sliding downhill into an energy valley you can't see into. A new model rebuilds the whole memory by sending waves walking across the network, and you can watch every step.

← All writing

Recalling a whole memory from a fragment, a face from a glimpse, a song from its first notes, is something brains do effortlessly. Machines do it too, with associative memories: store a set of patterns, then return the closest one from a partial or noisy cue.

The reigning models, from the classic Hopfield network to its modern attention-based descendants, all explain recall the same way: memories are valleys in an energy landscape, and recall is a slide downhill toward the nearest one. It works, but it's indirect. The dynamics have no closed-form trajectory, so you can say where recall ends up but not the path it took, which makes robustness and the dreaded "spurious states" hard to pin down.

A paper by Shikder, Liboni, Mináč, Budzinski and Muller takes a different route, resting on one technical fact: their network can be solved exactly.

01 / The shiftFrom energy descent to explicit trajectory

The model is a network of coupled complex-valued oscillators. A nonlinear change of variables turns its dynamics into a strikingly simple linear form:

x(t) = eKt x(0) the cue x(0) evolves by a single matrix exponential

That's the whole evolution. The connectivity matrix K is complex: its modulus sets the coupling strength along each edge, its argument sets a phase delay. Recall is no longer an iterative descent you can only simulate, it's an explicit linear evolution, read out in a single nonlinear step. And because the state at any time is something you can write down, you can interpret it directly.

02 / The pictureRecall is a sum of complex walks

Expanding that matrix exponential is where the intuition lives. The exponential is a sum of powers of K, and the entry of Kℓ collects every walk of length ℓ from one node to another across the network.

Because the edges are complex, each walk carries both an amplitude and a phase, the sum of the angles along its path, and walks combine by interference. Recall is then literally the network propagating the visible cue outward along all paths at once: walks that pass through a stored pattern arrive in phase and reinforce each other, rebuilding the occluded parts, while walks that would support a competing pattern arrive out of phase and cancel.

An occluded pixel is resolved not by its own missing value, but by the coherent walks that reach it from the parts of the cue still visible.

03 / The filtrationShort walks first, then longer ones

The walk picture makes recall something you can watch unfold. Short walks carry mostly local structure; longer walks carry progressively more global relationships. Collecting walks only up to some length m gives a truncated propagator, and increasing m produces an explicit filtration: the memory is rebuilt step by step, in plain view.

K(m)(t) = Σℓ=0m (tℓ/ℓ!) Kℓ keep walks up to length m; let m grow to recover the full propagator

This replaces the opaque descent of energy-based models with an inspectable reconstruction. You can stop at any walk length and see exactly how much of the image has come back, and where.

Because the dynamics amplify only the memory subspace and leave everything else stationary, there is nowhere for recall to settle that isn't a real memory.

04 / The constructionMemories as degenerate eigenmodes

How do you make the right walks reinforce? You choose K carefully. Each binary image is mapped to a phasor pattern; the patterns are stacked into a memory matrix and orthogonalized by a QR decomposition. The resulting directions are promoted to degenerate eigenmodes of K, all sharing a single eigenvalue, while every orthogonal direction carries none.

K = Q ( ε e−iφ I ) Q* stored patterns become eigenvectors sharing one eigenvalue

The consequence is clean. In-subspace walks grow with time while orthogonal contributions stay bounded and stationary. Spurious states are excluded by construction, not discouraged by tuning an energy surface, and retention is set by the spectrum of K rather than by an energy basin.

05 / The payoffHeavy occlusion, no spurious collapse

The results follow the picture. The network reconstructs MNIST digits at up to about 90% occlusion while staying near-perfect, where a Modern Hopfield Network (MHN) holds only to roughly 65%. It also generalizes to digits it never stored, consistently beating the MHN, and sustains accuracy as the number of stored memories grows.

✓

Robust. Phase, not magnitude, carries the signal, so recall survives heavy occlusion.

✓

Clean. No attractors live outside the stored patterns, so ambiguous inputs don't fall into spurious junk.

✓

Cheap. One linear operator plus a readout, matching the cost of a single attention step, no iterative unrolling.

06 / The clean breakWhy spurious states can't form

A spurious state is the classic failure of an energy-based memory: a final configuration that matches none of the stored patterns, a blended blob the network nonetheless settles into confidently. They appear because shaping an energy surface to hold many memories inevitably carves extra, unintended valleys between them. You can discourage these valleys, but in an energy model you can never quite be rid of them.

This network rules them out by construction rather than by tuning. Recall only ever amplifies directions inside the stored-memory subspace; everything orthogonal to it stays perfectly still. There is simply nowhere to settle that isn't a real memory, because no attractor exists outside the span of the stored patterns.

Spurious states aren't suppressed by a better-shaped landscape. The dynamics have nowhere else to go.

The sharpest test is a deliberately ambiguous cue: blend a "7" and a "2" into a mixture (1−α)·7 + α·2 and ask each network to resolve it. The MHN frequently collapses into a spurious blob for intermediate blends. The complex-walk network instead snaps to whichever genuine memory the cue leans toward, transitioning cleanly across the boundary and failing only at the perfectly symmetric midpoint, where it simply declines to commit.

07 / Why it mattersMemory that maps onto hardware

Beyond interpretability, the model is the kind of phase-based, linear computation that oscillator, photonic and analog hardware implement most naturally, suggesting a route to building these memories physically. The same eigenmode construction also extends past images: assign different eigenvalues to different memories and you can encode salience or recency directly into the spectrum.

And the marginal case at the exact "7"/"2" midpoint, where the network refuses to commit, echoes how perception wobbles at a category boundary, hinting at a tractable model of perceptual ambiguity.

08 / In a sentenceThe takeaways

▷

An associative memory built from complex oscillators has an exact closed-form recall: x(t) = eKtx(0).

▷

Recall is a sum of complex walks that interfere; coherent walks rebuild a memory, conflicting ones cancel.

▷

Ordering walks by length gives an inspectable filtration, replacing the opaque downhill slide of energy-based models.

▷

Storing memories as degenerate eigenmodes rules out spurious states by construction, survives ~90% occlusion, and maps naturally onto analog hardware.

"Complex walks over complex-valued neural networks solve image completion and memory association."
